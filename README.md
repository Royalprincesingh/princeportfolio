# 📱 Prince Singh - Developer Portfolio

## 🎯 Introduction
A modern, responsive developer portfolio showcasing Prince Singh's expertise in full-stack development, featuring smooth animations, interactive project showcase, and direct contact capabilities.

## ✨ Features
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Smooth animations and transitions using Framer Motion
- ✅ Interactive project showcase with detailed descriptions
- ✅ Contact form with email validation
- ✅ Smooth scrolling navigation
- ✅ Modern glassmorphism UI effects
- ✅ Social media links (GitHub, LinkedIn, Twitter)
- ✅ About section with skills overview
- ✅ Dark/Light theme support (if applicable)

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Git

### Frontend Setup
```bash
# Clone the repository
git clone https://github.com/royalprincesingh/princeportfolio.github.io.git
cd princeportfolio.github.io

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Backend Setup (if applicable)
```bash
# Navigate to backend directory
cd backend

# Install backend dependencies
npm install

# Start backend server
npm start
```

## 🚀 Deployment

### Deploy to Firebase (Recommended)
```bash
# Install Firebase CLI
npm install -g firebase-tools

# Login to Firebase
firebase login

# Initialize Firebase (if not done)
firebase init

# Deploy to Firebase Hosting
firebase deploy
```

### Deploy to Vercel
```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel

# For production deployment
vercel --prod
```

### Deploy to Netlify
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod --dir=dist
```
